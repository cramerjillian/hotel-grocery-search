Hotel-Grocery Search

About

Hotel-Grocery Search is a single-page application meant to support travel planning. I was inspired to make this tool by my experiences planning low-budget travel, specifically searching for lodging with necessities within walking distance.

Unfortunately, this is an inconvenient task using Google Maps--currently, one has to first search for hotels and manually perform a follow-up search for nearby grocery stores at each hotel result.

To fulfill this use-case, this tool uses Google APIs and libraries to perform a simultaneous search for two different establishment types with in a given city a proximity parameter included. Users may enter any U.S. city and a distance to find hotels within their preferred distance of a grocery store in that city.

How to Use:
  • Enter a U.S. city and state (abbreviated).
  • Enter a distance (in miles).
    • Represents the maximum distance you will accept between a hotel and grocery store in your search.

  • Use the interactable map and table to view your hotel-grocery store search results.
    • Click and drag to move the view of the map, CTRL+Zoom to zoom in.
    • Select rows of the search result table to view select hotel-grocery results.
    • Sort the result table by clicking any of the headers.
